4. Prompt for a BPMN diagram (Business Process Model and Notation)
This prompt constructs a model of user interaction with the system.

Prompt:
Based on the project data, create a BPMN diagram.
Please describe the key interaction steps:
Client → System Login
Selecting an Object → Starting a VR Tour
Submitting a Request → Contacting a Realtor
Discussing Terms → Concluding the Deal

Response should be in text format
