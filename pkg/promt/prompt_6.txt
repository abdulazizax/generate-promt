6. Промпт для оценки сроков и стоимости проекта
Этот промпт рассчитывает время разработки, состав команды и стоимость.
Промпт:
На основе функционального списка оцени сроки разработки, состав команды и стоимость проекта.
📌 Укажи:
Оптимальный состав команды: 
PM - 2200$
Teamlead - 2400$
Backend - 2200$
Frontend - 2200$
UX/UI - 2400$
QA - 2200$
DevOPS - 2700$ 
Оценку часов по модулям (Backend, Frontend, Дизайн, Тестировщик, DevOPS)
Итоговую стоимость (по ставке/160часов*кол-во часов*курс доллара)
Финансовый план (30% предоплата, оставшиеся платежи по месяцам)
=======
На основе предоставленного функционального списка оцени примерные сроки разработки, состав команды и общую стоимость проекта. 
Учитывай фиксированные месячные ставки (в долларах США) для каждой роли:

- Project Manager (PM): 2200
- Team Lead: 2400
- Backend Developer: 2200
- Frontend Developer: 2200
- UX/UI Designer: 2400
- QA Engineer: 2200
- DevOps Engineer: 2700

Определи:
1) Сколько месяцев (months) и сколько специалистов (count) нужно по каждой роли, 
2) Сумму (sum) для каждой роли (sum = months * monthlySalary * count),
3) Часы (hours), почасовую ставку (hourlyRate) и итоговую стоимость (cost) для модулей (например, Backend, Frontend, Design, Testing, DevOps),
4) Финансовый план (financialPlan) с полями:
   - "prepaymentPercent" (процент предоплаты),
   - "prepayment" (сумма предоплаты),
   - "monthlyPayments" — массив, в котором каждый элемент соответствует сумме за определённый месяц (например, [4000, 4500, 3700] если 3 месяца),
   - "totalProjectCost" — общая сумма проекта.

Верни ответ строго в формате JSON (без тройных кавычек и пояснений). Структура:
- "team": массив объектов с полями "role", "count", "months", "monthlySalary", "sum".
- "modules": массив объектов с полями "moduleName", "hours", "hourlyRate", "cost".
- "financialPlan": объект с полями 
  "prepaymentPercent", 
  "prepayment", 
  "monthlyPayments" (массив чисел), 
  "totalProjectCost".
