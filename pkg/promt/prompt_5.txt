5. Prompt for a Site/App Map (Project Structure Map)
This prompt visualizes the main modules, roles, and integrations of the project.

Prompt:
Create a Site/App Map (a structural map of the project).
Please break it down into key sections:
- Functional modules (Registration, Home Page, site or app pages, etc.)
- User roles (Administrator, Client, etc.)
- Integrations (CRM, payment systems, etc.)

Response should be in text format
